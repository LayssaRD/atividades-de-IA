## 🧠 Atividades Realizadas

### 1. Experimentos DBSCAN — Clustering em Dados Complexos

Aplicação do algoritmo **DBSCAN** para agrupamento de dados com formatos não lineares.

**Principais pontos:**
- Identificação de clusters baseados em densidade
- Detecção de ruídos (outliers)
- Comparação com K-Means
- Análise de dados complexos onde centroides não funcionam bem

---

### 2. Segmentação RFM — Pipeline com Olist Brasil

Construção de um pipeline completo de análise de clientes utilizando a metodologia **RFM (Recência, Frequência e Valor Monetário)**.

**Principais etapas:**
- Tratamento do dataset Olist Brasil
- Cálculo das métricas RFM
- Aplicação do algoritmo K-Means
- Definição de segmentos de clientes:
  - Campeões
  - Em Risco
  - Inativos
  - Novos/Recentes

**Insights gerados:**
- Identificação do segmento Em Risco como maior gerador de receita
- Oportunidades de retenção e fidelização
- Estratégias de marketing baseadas em comportamento

---

### 3. LLM Local — Gemma + Análise de Clientes

Uso de um modelo de linguagem local (**Google Gemma 3 4B via Ollama**) para interpretação dos resultados de análise de clientes.

**Principais pontos:**
- Execução 100% local (sem APIs externas)
- Interpretação automática de clusters
- Geração de estratégias de marketing
- Criação de e-mails personalizados por segmento
- Comparação entre modelos (Gemma 1B vs 4B)
